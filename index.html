Software Tools in GNU ARM Assembler

  • Implementierung der Software Tools von Kerningham und Plauger in GNU ARM Assembler
  • Start-Projekt, um strukturierte Maschinensprache zu entwickeln
  • Laufen auf Raspberry Pi 3
@include(copy.x)
  • Erstes Programm kopiert die Eingabe in die Ausgabe
  • Durch Anwahl von copy.x im @include-Tag kann in die Beschreibung von copy gewechselt werden
@include(count-chars.x)
  • Zählt alle Zeichen der Eingabe und gibt die Anzahl aus
  • Durch Anwahl von count-chars.x im @include-Tag kann in die Beschreibung von count-chars gewechselt werden
@include(count-lines.x)
  • Zählt alle Zeilen der Eingabe und gibt die Anzahl aus
  • Durch Anwahl von count-lines.x im @include-Tag kann in die Beschreibung von count-lines gewechselt werden
@include(count-words.x)
  • Zählt alle Wörter der Eingabe und gibt die Anzahl aus
  • Durch Anwahl von count-words.x im @include-Tag kann in die Beschreibung von count-words gewechselt werden
@include(un-tab.x)
  • Wandelt Tabulatoren in Freizeichen um
  • Durch Anwahl von un-tab.x im @include-Tag kann in die Beschreibung von un-tab gewechselt werden
